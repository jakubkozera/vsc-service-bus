# Service Bus Explorer (VS Code)

Manage Azure Service Bus namespaces, queues, topics, subscriptions, rules and messages from VS Code.

A modern rewrite of the classic Windows-only ServiceBusExplorer using `@azure/service-bus` and `@azure/identity`.

## Build

```
npm install
npm --prefix webviews install
npm run build
```

Press `F5` to launch the Extension Development Host.
